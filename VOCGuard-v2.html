🔍 Assessor Quality & Calibration Report

Understanding This Report

This report analyses assessor performance to identify calibration issues, inconsistencies, and quality concerns. Rubber-stamping is flagged when an assessor's average score is significantly higher than the fleet mean, suggesting insufficient rigour. Calibration Index compares each assessor's score distributions against the fleet average for the same activities — a high index suggests over-lenient or over-strict scoring.

⚙️ Settings & Configuration

📊 Google Sheets Integration

Connect to a Google Apps Script web app to automatically sync every saved assessment to Google Sheets for trending, reporting, and offline backup.

Setup Instructions:
1. Open script.google.com — create a new project named "VOCGuard".
2. Download the Apps Script code using the button below and paste it into the script editor (Code.gs).
3. Click Deploy → New Deployment → Web App. Execute as: Me. Who has access: Anyone.
4. Authorise the permissions, then copy the deployment URL and paste it above.
5. Click Test Connection — if successful you'll see a green confirmation.

🖥️ Display Settings

Switch off demo data when you're ready to use VOCGuard for live assessments. The register will show only real saved assessments.
